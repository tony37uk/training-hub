Azure Fundamentals — AZ-900

Everything you need to prepare for the AZ-900 certification — the perfect starting point for anyone new to cloud computing.

AZ-900 is Microsoft's entry-level Azure certification. It validates foundational knowledge of cloud concepts and Azure services — no technical background required. It's the recommended first cert for anyone starting a cloud career or moving into IT roles that involve Azure.

🎓 Exam cost: £115 (discounts often available via Microsoft promotions and exam vouchers through employer schemes — ask your manager!).
