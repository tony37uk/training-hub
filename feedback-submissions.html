Follow these 4 steps to connect a free persistent database. Takes about 5 minutes.

  1. Go to supabase.com → sign up free → create a new project (pick any region)
  2. Once the project loads, click SQL Editor in the left sidebar and run this query:
    create table feedback (
      id bigint generated always as identity primary key,
      name text not null default 'Anonymous',
      type text not null default 'General',
      message text not null,
      actioned boolean not null default false,
      created_at timestamptz not null default now()
    );
    alter table feedback enable row level security;
    create policy "anon_all" on feedback for all to anon using (true) with check (true);
  3. Go to Project Settings → API — copy your Project URL and anon / public key
  4. Open index.html and feedback-submissions.html in a text editor — replace YOUR_PROJECT_REF and sb_publishable_4Lnq5dM-5j8K6lDtunhcFw_pSDyc64O with your copied values, then push to GitHub
