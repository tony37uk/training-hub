Getting Started with OpenClaw

⏱️ 20–30 min read 🟡 Intermediate 🖥️ Self-hosted on your machine

OpenClaw is an open-source personal AI assistant that runs on your own machine and connects to the chat apps you already use — WhatsApp, Telegram, Discord, Slack, and more. Your data stays yours. The AI actually does things.

💡 In one sentence: OpenClaw is the AI that actually does things — it clears your inbox, sends emails, manages your calendar, browses the web, runs code, and builds its own new skills, all from a chat message.

What Is OpenClaw?

OpenClaw was created by Peter Steinberger and rapidly grew into one of the most talked-about open-source AI projects of 2026. It's a persistent agent that lives on your machine, has memory of your context and preferences, and can take real actions on your behalf autonomously.

What OpenClaw Can Do

Communication

  • ✓Read, write, and send emails via Gmail and other mail providers
  • ✓Manage your calendar — check availability, create and update events
  • ✓Send and receive messages across WhatsApp, Telegram, Discord, Slack, Signal, and iMessage
  • ✓Draft responses, summarise threads, and triage your inbox

Computer & File Access

  • ✓Read and write files on your machine
  • ✓Run shell commands and scripts
  • ✓Control the browser — navigate pages, fill forms, extract data from any website
  • ✓Execute code in Python, JavaScript, and other runtimes

Skills & Automation

  • ✓Extend with community skills from ClawHub (clawhub.ai)
  • ✓Build custom skills by describing what you want — it writes the skill itself
  • ✓Set up cron jobs, reminders, and background tasks that run proactively
  • ✓Connect to 50+ integrations including GitHub, Obsidian, Spotify, and VirusTotal
🔥 Self-building: One of OpenClaw's most remarkable features is that it can write and install its own new skills based on your requests — effectively extending its own capabilities through conversation.

💰 Model costs: OpenClaw uses your own AI model subscription or API key — it does not include model access. If you have a Claude Max subscription, you can use that. The Anthropic API, OpenAI API, or local models like MiniMax are all supported.

Installation

1

Option 1 — One-line installer (recommended)

The fastest way to get started. Open a terminal and run:

# macOS and Linux curl -fsSL https://openclaw.ai/install.sh | bash # Windows (PowerShell) iwr -useb https://openclaw.ai/install.ps1 | iex

The installer handles Node.js and all other dependencies automatically.

2

Option 2 — npm install (if you already have Node 22+)

npm i -g openclaw
3

Option 3 — From source (full control)

git clone https://github.com/openclaw/openclaw.git cd openclaw && pnpm install && pnpm run build

First Run — The Onboarding Wizard

Once installed, run the onboarding wizard:

openclaw onboard --install-daemon

The wizard will walk you through:

1

Authentication

Connect your AI model API key — Anthropic, OpenAI, or a local model.

2

Channel Setup

Connect your preferred chat app (WhatsApp, Telegram, Discord etc.).

3

Persona Setup

Name your assistant and give it initial personality and context about you and your work.

4

Open the Control UI

Run openclaw dashboard to open the browser-based Control UI at http://127.0.0.1:18789/ — the fastest way to have your first conversation without needing a chat app first.

Your First Conversations

Exploring what it can do

  • →"What can you do?" — overview of current skills
  • →"What do you know about me so far?" — review what it has remembered
  • →"Show me my schedule for today" — test calendar access
  • →"Summarise the last 10 emails in my inbox" — test email access

IT professional use cases

  • →"Check my GitHub repos for any open pull requests and give me a summary"
  • →"Write a Python script that exports my Intune device list to CSV" — it will write and run the code
  • →"Every morning at 8am, send me a briefing covering my emails, calendar, and any alerts"
  • →"Research Azure pricing changes in 2026 and save a summary to a file on my desktop"
💡 Tip: OpenClaw works best when you treat it like a capable colleague rather than a search engine. Give it context, tell it your preferences, and delegate whole tasks rather than single questions.

Skills — Extending OpenClaw's Capabilities

Installing community skills

Browse and install skills from ClawHub (clawhub.ai). Simply tell OpenClaw:

"Install the Todoist skill from ClawHub"

Building your own skills

Ask OpenClaw to build a custom skill by describing what you want:

  • →"Create a skill that checks my Intune device compliance daily and messages me a summary"
  • →"Build a skill that monitors my GitHub Actions and notifies me of failed builds"
  • →"Make a skill that pulls my calendar for the week and formats it as a plain text briefing"

OpenClaw will write the skill definition, install it, and begin using it — all within the conversation.

Privacy, Security, and Safe Use

What stays on your machine

  • ✓All memory, context, and conversation history is stored locally by default
  • ✓Your data does not pass through OpenClaw's servers — only through your chosen AI model's API
  • ✓Skills and configuration files are stored in your home directory and are fully readable

Important safety practices

  • !Review actions before they are taken — use the approval prompts for emails and file operations
  • !Start with read-only tasks before granting write access to sensitive systems
  • !Check the VirusTotal integration to scan skills before installing from third parties
  • !Keep OpenClaw updated — security patches are released frequently
⚠️ Prompt injection awareness: Like all AI agents, OpenClaw can potentially be manipulated by malicious content it reads — for example, a webpage or email with hidden instructions. Review agent actions in sensitive contexts and never store credentials inside OpenClaw's memory.

Quick Reference — Do's and Don'ts

✅ Do this

  • Start with the Control UI before connecting chat apps
  • Give it context about yourself during onboarding
  • Begin with read tasks before write/send tasks
  • Use approval prompts for emails and file changes
  • Browse ClawHub for ready-made skills first
  • Keep OpenClaw updated — it ships fast

❌ Avoid this

  • Storing passwords or API keys in OpenClaw's memory
  • Giving it write access to production systems on day one
  • Skipping the approval step for bulk email actions
  • Installing skills from unknown sources without scanning
  • Ignoring update notifications — security patches matter
  • Running as a root/admin user unnecessarily
